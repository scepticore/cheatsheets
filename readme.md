# Cheatsheet creator

A simple cheatsheet creator with Markdown-Inspired Syntax,
designed for printed A4 cheatsheets

## Features

- Multi-Column-Cheatsheet creator (1-5)
- Font-Size-Picking between 6Pt and 12Pt
- Live-Preview in Browser (like Overleaf)
- PDF-Export
- Group-Working

## Syntax

| `code`                   | Description           |
|--------------------------|-----------------------|
| `# Text`                 | Heading 1             |
| `## Text`                | Heading 2             |
| `### Text`               | Heading 3             |
| `#### Text`              | Heading 4             |
| `##### Text`             | Heading 5             |
| `###### Text`            | Heading 6             |
| `***Text***`             | ***Bold and italic*** |          
| `**Text**`               | **Bold text**         |
| `*italic text*`          | *Italic text*         |
| `-duregstriche-`         | Crossed text          |
| `- Listitem`             | List, Level 1         |
| `-- Listitem`            | List, Level 2         |
| `--- Listlistitem`       | Listitem              |
| `[img src=image.jpg]`    | Image                 |
| `` `code example` ``     | Code                  |
| ` ```multiline code``` ` | Multiline Code        |
| `---`                    | Horizontal line       |
